# Fire Extinguishing Status Prediction

## Overview
Classification model to predict fire extinguishing status in buildings based on acoustic wave parameters.

## Tools
- Python
- Pandas
- Scikit-learn

## Result
- Evaluated AdaBoost, Random Forest, and Gradient Boosting classifiers
- Best model accuracy around 96% on test split
- F1-score reported around 0.95-0.96 in classification outputs

## Business Impact
Helps evaluate acoustic-based fire suppression strategies and supports safer, data-driven building safety system design.
