# Human Activity Detection Soft Sensor

## Overview
Machine learning project to detect human activities from mobile gyroscope and accelerometer signals using KU-HAR dataset features.

## Tools
- Python
- Pandas
- Scikit-learn

## Result
- Validation accuracy up to 96.13% (Random Forest)
- Test accuracy around 95.48% (Random Forest)
- KNN baseline around 90.66% test accuracy

## Business Impact
Supports fast and low-cost activity monitoring use cases, such as health tracking and context-aware mobile applications.
