# Mill Outlet Temperature Soft Sensor

## Overview
Soft sensor development project to predict Mill A Unit 3 outlet temperature from historical plant process signals.

## Tools
- Python
- Pandas
- Scikit-learn
- TensorFlow/Keras

## Result
- AdaBoost/ExtraTrees style regressors produced test R2 up to 0.768
- Validation R2 reached around 0.714
- Best classical model runs showed test MAE around 0.379 and validation MAE around 0.441

## Business Impact
Enables earlier temperature prediction for process control, improving operational stability and reducing manual intervention.
