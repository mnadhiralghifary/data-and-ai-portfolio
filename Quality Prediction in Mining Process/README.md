# Mining Process Quality Prediction

## Overview
Regression project to predict mining process quality using process variables from flotation plant data.

## Tools
- Python
- Pandas
- Scikit-learn

## Result
- Compared AdaBoost Regressor and KNN Regressor approaches
- AdaBoost achieved test MSE around 0.090 and validation MSE around 0.097
- KNN showed test MAE around 0.163 and test MSE around 0.111

## Business Impact
Supports process optimization by enabling earlier quality estimation and reducing operational variability.
