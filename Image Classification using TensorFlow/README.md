# Image Classification with TensorFlow

## Overview
Computer vision project for image classification using a pre-trained TensorFlow graph and label mapping workflow.

## Tools
- Python
- TensorFlow
- NumPy

## Result
- Built end-to-end image inference pipeline (graph loading, label mapping, and prediction)
- Generated class prediction outputs from test images
- Experiment artifacts organized in model and output directories

## Business Impact
Provides a reusable baseline for fast visual inspection and automated object recognition workflows.
